Guia de Tags HTML

  • <p>
    Define um parágrafo de texto.
  • <h1> até <h6>
    Títulos e subtítulos, onde h1 é o mais importante e h6 o menos importante.
  • <a>
    Cria links para outras páginas ou recursos. Use href="" para definir o destino.
  • <img>
    Insere imagens. Requer src="" para o caminho da imagem e alt="" para descrição.
  • <ul> e <li>
    Lista não ordenada (ul) e itens da lista (li).
  • <ol>
    Lista ordenada (numerada). Use com <li> para os itens.
  • <div>
    Container genérico para agrupar elementos e criar layouts.
  • <span>
    Container inline para texto e elementos pequenos.
  • <table>, <tr>, <td>
    Cria tabelas: table (tabela), tr (linha), td (célula).
  • <form>
    Cria formulários para entrada de dados.
  • <input>
    Campo de entrada em formulários. Tipos: text, email, password, etc.
  • <button>
    Cria botões clicáveis.
  • <textarea>
    Campo de texto multilinha para formulários.
  • <br>
    Quebra de linha.
  • <hr>
    Linha horizontal para separação de conteúdo.
  • <strong>
    Texto em negrito, usado para destacar conteúdo importante.
  • <em>
    Texto em itálico, usado para dar ênfase.
  • <header>
    Define o cabeçalho de uma página ou seção.
  • <footer>
    Define o rodapé de uma página ou seção.
  • <nav>
    Define uma seção de navegação com links.
  • <main>
    Define o conteúdo principal da página.
  • <blockquote>
    Usado para citações longas de outros textos.
  • <cite>
    Define o autor ou fonte de uma citação.

💡 Lembre-se:

  • A maioria das tags precisa ser fechada: <tag>conteúdo</tag>
  • Algumas tags são "vazias" e não precisam fechar: <img>, <br>, <hr>
  • Use indentação para organizar seu código
  • Mantenha as tags organizadas e alinhadas

Antes de começarmos, algumas coisas importantes que você precisa saber:

🤔 O que é Programação Web?

Programação web é como construir uma casa na internet! Cada site que você visita foi construído por alguém usando códigos especiais.

🏗️ O que é HTML?

HTML (HyperText Markup Language) é como os blocos de construção dessa casa. É a linguagem que usamos para criar a estrutura de todas as páginas web que você vê na internet!

📝 Como Funciona?

No HTML, usamos "tags" (marcações) para dizer ao computador o que cada parte do site significa. Por exemplo:

  • <p> significa "isto é um parágrafo"
  • <h1> significa "isto é um título importante"
  • <img> significa "aqui vai uma imagem"

🏗️ Estrutura Básica do HTML

Anatomia de uma Tag HTML

<tag atributo="valor">
    conteúdo
</tag>
  • 🔍 tag: Nome do elemento HTML (p, div, span, etc)
  • 🎨 atributo: Propriedade adicional (class, id, style, etc)
  • 📝 valor: Valor do atributo
  • 📄 conteúdo: O que vai dentro da tag

Estrutura de uma Página HTML

<!DOCTYPE html>
<html>
    <head>
        <title>Título da página</title>
    </head>
    <body>
        Conteúdo da página
    </body>
</html>
  • 📄 <!DOCTYPE html>: Declara que é um documento HTML5
  • 🌐 <html>: Elemento raiz da página
  • 🧠 <head>: Informações sobre o documento
  • 👀 <body>: Conteúdo visível da página
